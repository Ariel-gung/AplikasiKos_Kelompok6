# AplikasiKos - Kelompok 6
## Mata Kuliah: Pemrograman Berorientasi Objek (PBO)
### Institut Teknologi Kalimantan

Aplikasi manajemen kos-kosan yang memungkinkan pencari kos untuk mencari dan memesan kamar, serta pemilik kos untuk mengelola kamar dan pemesanan.

## Struktur Proyek

```
AplikasiKos_Kelompok6/
├── Ariel_04231012.txt                      # File latihan Git
├── UML_AplikasiKos.pdf                     # Diagram UML (PDF)
├── UML_Diagrams.html                       # Diagram UML (HTML Source)
├── app/
│   └── src/main/
│       ├── java/com/example/aplikasikos/
│       │   ├── DataModels.kt               # Data Model dengan Enkapsulasi
│       │   ├── MainActivity.kt             # Activity Utama
│       │   └── ...
│       └── res/layout/
│           ├── activity_login.xml          # Layout Halaman Login
│           └── activity_main.xml           # Layout Halaman Beranda
└── ...
```

## Fitur Utama
- **Cari Kamar**: Pencari kos dapat mencari kamar berdasarkan lokasi dan harga
- **Pesan Kamar**: Pencari kos dapat memesan kamar yang tersedia
- **Kelola Kamar**: Pemilik kos dapat mengelola data kamar
- **Kelola Pemesanan**: Pemilik kos dapat mengelola pemesanan masuk
